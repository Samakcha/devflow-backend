# DevFlow Backend

A production-grade backend project built to master modern backend engineering concepts.

## Current Features

- Express.js application architecture
- Modular routing, controllers, and services
- MongoDB Atlas integration with Mongoose
- User Registration
- User Login
- Password hashing with bcrypt
- JWT Authentication
- Health Check API

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- Redis (Upcoming)
- BullMQ (Upcoming)
